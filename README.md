# 🎵 Player de Música

Um projeto de player de música interativo, Ele simula um aplicativo de streaming real, realizando buscas e reproduzindo prévias de músicas diretamente do navegador.

##  Funcionalidades
* **Busca Dinâmica:** Integração com a API do iTunes para pesquisar faixas reais.
* **Reprodução de Áudio:** Toca os 30 segundos de prévia musical disponibilizados pela API.
* **Controles de Navegação:** Botões de avançar e voltar que permitem alternar perfeitamente entre as próximas músicas ou faixas anteriores retornadas pela API, do artista em questão ou músicas.
* **Barra de Progresso Interativa:** Permite que o usuário clique na barra de tempo para pular diretamente para um trecho da música.
* **Atualização em Tempo Real:** A interface atualiza automaticamente a capa do álbum, nome da música e nome do artista conforme a faixa selecionada.
* **Design Responsivo:** Utilização de Media Queries no CSS para garantir a melhor estilização e usabilidade tanto em notebooks quanto em celulares.

## 🛠️ Tecnologias Utilizadas
* **API do iTunes:** Endpoint fornecedor dos dados musicais e arquivos de áudio.
* **HTML5 & CSS3:** Estrutura e estilização do player convencional.
* **JavaScript:** Responsável por toda a lógica de funcionamento da aplicação.
* **jQuery:** Utilizado para as requisições de busca na API.
* **Recursos Externos:** Google Fonts (tipografia) e Font Awesome (ícones).

## 💻 Como executar
Experimente o player diretamente do navegador acessando o link 
https://beeflobers.github.io/Pink-Kitty-Player/
